Explain the difference between a GET and POST method in forms
GET method is used when a form data information is added and included into the URurl, when this is run, all details of the form is shown in the url.
POST method is used when submitting sensitive info (passwords, messages, etc.), it does not include sensitive info inside the url
example GET is writing your message on the outside of the envelope  where anyone can see it. while POST is putting your message inside the envelope, it is safe,private and hidden
